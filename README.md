# Tech blog starter-kit with vue3, nuxt3 and Contentful

- Vue3
- Nuxt3
- Vite
- Volar
- TypeScript
- Yarn 3
  - use .yarn/cache

# init

- use Node.js v16.14.2^
- install VSCode extenstion `Volar`
- `yarn install`

# Start dev server

```
yarn dev -o
```

But, does not work. see [issue #1](https://github.com/longrun/tech-blog-vue3-nuxt3/issues/1)

# build and start

```
yarn build
yarn start
```
